 1. Movies API
🔹 Popular Endpoints:
GET /movie/popular → Trending movies

GET /movie/top_rated → Top-rated by users

GET /movie/upcoming → Upcoming releases

GET /movie/now_playing → Currently in theaters

GET /movie/{movie_id} → Full movie details

GET /movie/{movie_id}/videos → Trailers

GET /movie/{movie_id}/images → Posters, backdrops

GET /movie/{movie_id}/credits → Cast & Crew

GET /movie/{movie_id}/recommendations → Similar movies

GET /movie/{movie_id}/reviews → User reviews

🧾 Key Fields in Movie Detail:
Field	Description
title	Movie title
overview	Summary/description
poster_path	URL to the movie poster
backdrop_path	Wide background image
release_date	Date of release
vote_average	Average rating
vote_count	Number of votes
runtime	Length in minutes
genres	List of genres (with IDs and names)
production_companies	Studios that made the movie
spoken_languages	Languages used in the film
original_language	Original language of the movie
status	e.g. Released, In Production
tagline	Short promotional quote
budget / revenue	Budget and box office numbers
homepage	Official website URL
videos	Trailers & behind the scenes (YouTube keys)

📺 2. TV Shows API
GET /tv/popular

GET /tv/{tv_id} → Similar to movies, includes:

Seasons & episodes

Air dates

Networks

Episode runtime

Trailers

👤 3. People (Actors, Directors)
GET /person/popular

GET /person/{person_id} → Get actor/director details

GET /person/{person_id}/movie_credits → Films they acted in or directed

Key Fields:
name, biography, birthday, deathday

known_for_department (e.g. Acting, Directing)

place_of_birth

profile_path (headshot image)

gender

combined_credits (movies + TV roles)

🔍 4. Search API
GET /search/movie → Search by title

GET /search/person → Search actors/directors

GET /search/multi → Search all categories at once

Supports:

Query text

Year filters

Pagination

Language/localization

🧠 5. Genres
GET /genre/movie/list → Get genre IDs and names

Use this to map genre IDs to human-readable names (e.g., 28 → "Action")
