Important:

The HorizontalTVShowCard component uses the "Schkorycza Regular" font.

Please place the following font files in this directory:
- SchkoryczaRegular.woff2
- SchkoryczaRegular.woff

You can download the font from its official source or a reputable font provider.

If the font files are not available, the component will fall back to the system's sans-serif font.
